Конструктор Инструкции О команде

Как пользоваться этим приложением?

Главные предметы в параграфе этого конструктора:

  • Ссылка на картинку
  • Заголовок
  • Текст параграфа
  • Выберите фон
  • Выравнивание картинки
  • Размер картинки
  • Добавить
  • Удалить всё
  • Удалить компонент
  • Удалить последний компонент
  • Скачать

Подробный разбор:

  • Поле "Ссылка на картинку" - для добавления ссылки на картинку которую нужно добавить.
  • Поле "Заголовок" - для выбора заголовка абзаца
  • Поле "Текст параграфа" - для добавления текста
  • Поле "Выберите фон" - для выбора цвета фона.
  • Поле "Выберите шрифт" - для выбора семейства шрифтов текста.
  • Поле "Выравнивание картинки" - для выбора положения картинки относительно текста.
  • Поле "Размер картинки" - для выбора размера изображения.
  • Кнопка "Добавить" - для вставки компонента в область конструктора.
  • Кнопка "Удалить всё" - удаляет всё что вы добавили.
  • Кнопка "Удалить компонент" - удаляет компонент по которому вы кликнете дважды.
  • Кнопка "Удалить последний компонент" - удаляет последний элемент который вы добавили.
  • Кнопка "Скачать" - позволяет скачать сконструированный сайт.
